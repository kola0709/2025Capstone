# 🌟 초신성 | **목소리의 모양**  
**Flask 및 머신러닝 기반 수어 통역 및 학습 플랫폼**

---

## 🎬 프로젝트 개요

> **목소리의 모양**은 Mediapipe와 LSTM 등 다양한 딥러닝 기반 기술을 활용하여  
> 사용자의 수어 동작을 실시간으로 인식하고 번역하는 **수어 통역 및 학습 플랫폼**입니다.

이 플랫폼은 수어 번역 기능 외에도  
- 📘 **수어 단어장 기능**
- 🎮 **수어 게임 학습 기능**
- 🎨 **커스텀 수어 등록 기능**  
을 통해 사용자에게 **학습, 실습, 실생활 적용**까지 가능한 통합적 수어 경험을 제공합니다.

---

## 🧠 핵심 기능

| 기능 구분 | 설명 |
|-----------|------|
| 🤖 **수어 번역기** | Mediapipe, LSTM 등을 이용한 실시간 수어 인식 및 번역 |
| 📘 **단어장** | 학습용 수어 모음 제공 |
| 🎮 **수어 퀴즈** | 게임을 통해 수어 학습 확인 |
| 🧩 **커스텀 수어 등록** | 나만의 수어 정의 및 저장 |
| 🌐 **플랫폼 제공** | 웹 기반 서비스 제공 (크로스 플랫폼 지원) |

---

## 🔧 기술 스택 및 구조

- **👨‍💻 개발 환경**  
  - Windows 11 (24H2), macOS Sequoia, Ubuntu 22.04 LTS

- **🧰 개발 도구**  
  - Visual Studio Code, MySQL Workbench

- **📚 사용 언어**  
  - Java, JavaScript, Python

- **🖥️ 주요 프레임워크/라이브러리**  
  - `React` (프론트엔드)  
  - `Spring Boot` (백엔드 API 및 회원관리)  
  - `Flask` (수어 예측 모델 API 서버)  
  - `Mediapipe`, `LSTM` (손동작 인식 및 수어 예측)
